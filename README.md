Projeto realizado para a quinta etapa do processo seletivo da Apple Developers Academy junto ao SENAC.
O desafio consistia em criar um sistema interativo utilizando Swift, baseado em um mundo de fantasia e/ou um RPG.

O projeto que desenvolvi foi inspirado em um mundo de magia, onde o usuario pode visualizar o livro de magias, aprender um feitico novo, adicionar um novo feitico, entrar em uma missao e ver seus feiticos aprendidos.
Utilizei os principais conteúdos aprendidos durante o workshop, como: Arrays, condicionais, funções, loops entradas do usuario e outros.
